# Panduan Upload ke GitHub & Mendapatkan Link

Untuk membagikan Web Ucapan Lebaran ini kepada orang lain melalui link, cara termudah dan gratis adalah menggunakan **GitHub Pages**. Ikuti langkah-langkah berikut:

## Cara 1: Menggunakan Website GitHub (Sangat Mudah)
Gunakan cara ini jika Anda belum pernah memakai aplikasi Git di komputer Anda.

1. Buka [GitHub.com](https://github.com/) dan pastikan Anda sudah **Login** / **Sign in**.
2. Klik tombol **New** (berwarna hijau) di pojok kiri atas untuk membuat *Repository* baru.
3. Beri nama repository, misalnya: `ucapan-lebaran-2026`.
4. Pastikan opsi **Public** terpilih (agar web bisa diakses orang lain).
5. Scroll ke bawah dan klik tombol **Create repository**.
6. Pada halaman selanjutnya, temukan tulisan *"...or create a new repository on the command line"* dan klik tautan **uploading an existing file** (tepat di bawah kalimat *"get started by creating a new file..."*).
7. Buka *File Explorer* di komputer Anda dan buka folder:  
   `e:\000 Kanwil Lampung\koding\kartu_ucapan_lebaran_2026`
8. **Drag and Drop** (Tarik dan Lepas) seluruh isi folder tersebut (file `index.html`, folder `css`, `js`, `assets`) ke dalam kotak upload di halaman GitHub tadi.
9. Setelah semua file selesai di-upload, klik tombol hijau **Commit changes**.

### Mengaktifkan Link Publik (GitHub Pages)
1. Setelah file berhasil masuk ke repository, klik tab **Settings** (Ikon Gir) di bagian atas halaman repository Anda.
2. Di menu sebelah kiri, cari bagian *Code and automation*, lalu klik **Pages**.
3. Di bawah tulisan *Build and deployment* -> *Source*, pilih **Deploy from a branch**.
4. Di bagian *Branch*, klik dropdown yang bertuliskan *None*, lalu pilih branch **main** (atau **master**).
5. Klik tombol **Save**.
6. **Selesai!** Tunggu sekitar 1-2 menit. Link web Anda akan muncul di bagian atas halaman **Pages** tadi, yang biasanya formatnya seperti ini:  
   `https://[username-github-anda].github.io/ucapan-lebaran-2026/`

## Cara Tambahan: Vercel / Netlify (Alternatif Instan)
Jika merasa GitHub Pages terlalu rumit, Anda bisa langsung mendapatkan link instan dengan menggunakan Netlify:
1. Buka [app.netlify.com/drop](https://app.netlify.com/drop)
2. Anda cukup melakukan Drag & Drop folder `kartu_ucapan_lebaran_2026` ke dalam lingkaran di halaman web tersebut.
3. Link akan langsung jadi dalam hitungan detik!

Bagikan link yang sudah Anda dapatkan kepada keluarga dan rekan-rekan. Selamat mencoba!
